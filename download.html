# 下载

在下载安装前请确认您的设备具备以下条件：

  • 系统版本至少为 Android 6.0（或 API >= 23 的兼容系统）
  • 能够访问网络，并允许后台网络权限
  • 允许应用显示悬浮窗
  • 系统不会随意杀掉任意后台服务，否则悬浮窗可能很容易被关闭（参见 Don't kill my app）
